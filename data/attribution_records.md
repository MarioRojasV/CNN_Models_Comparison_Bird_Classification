# Image Dataset Attribution Records

This directory contains the photographic assets utilized for the convolutional classification models. The dataset consists of 250 images for *Turdus grayi* and 250 images for *Quiscalus mexicanus* with 10 additional flying photos for each species, curated and filtered under strict Creative Commons (CC) guidelines.

To prevent numbering conflicts (as files for both species are indexed from 1 to 250), the complete legal metadata, author attributions, source URLs, and specific CC license records have been systematically partitioned by taxonomic class:

## Dataset Divisions & Compliance Logs

* 🐦 **[*Turdus grayi* (Yigüirro/Mirlo pardo/Clay-colored thrush) Attribution Records](turdus_grayi_attributions.md)**
  Contains the detailed metadata and TASL table for the 250 images corresponding to the Clay-colored Thrush class.

* 🐦‍⬛ **[*Quiscalus mexicanus* (Zanate grande/Great-tailed grackle) Attribution Records](quiscalus_mexicanus_attributions.md)**
  Contains the detailed metadata and TASL table for the 250 images corresponding to the Great-tailed Grackle class.
